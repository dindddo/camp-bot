📖 MCP 공식 문서 📖 MCP 오픈 표준 소개

MCP = USB-C. 다양한 서비스를 하나의 규격으로 AI에 연결합니다.

HTTP 방식 (클라우드 서비스)

claude mcp add --transport http notion https://mcp.notion.com/mcp

stdio 방식 (로컬 실행)

claude mcp add --transport stdio fetch -- npx -y @anthropic-ai/mcp-fetch@latest

관리 명령어

claude mcp list
📖 인기 MCP 서버 📖 MCP 서버 목록 (GitHub) 🔗 Connectors 설정
